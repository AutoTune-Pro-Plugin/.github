## 🚀 Overview
Auto-Tune Pro for Windows is the flagship pitch-correction and vocal-tuning plugin from Antares, designed for producers, mixing engineers, and vocalists who want polished, in-tune performances. It combines fast, low-latency Auto mode for real-time correction during tracking and live monitoring with a precise Graph mode for note-by-note graphical editing of pitch and timing. Because it loads as a plugin inside a host DAW, Auto-Tune Pro fits naturally into your existing workflow whether you are recording a lead vocal, sculpting harmonies, or adding a stylized tuning effect.

## ✨ Features
- **Real-time Auto mode** — Correct pitch on the fly with adjustable retune speed, from transparent tuning to the classic hard-tuned vocal effect.
- **Graphical Graph mode** — Edit individual notes, pitch curves, and timing by hand for detailed, natural-sounding results on important passages.
- **Natural or creative tuning** — Dial in subtle, invisible correction or push settings for bold, stylized vocal sounds.
- **Key and scale awareness** — Set the song key and scale so corrections snap musically to the notes you intend.
- **Works inside your DAW** — Runs as a plugin in popular hosts, so you can automate parameters and revise takes non-destructively.

## 🎯 Getting Started
1. Insert Auto-Tune Pro as a plugin on the vocal channel in your DAW.
2. Set the song key and scale so corrections follow the right notes.
3. Use Auto mode for instant real-time tuning and adjust the retune speed to taste.
4. Switch to Graph mode when you want to fine-tune specific notes, curves, or timing by hand.
5. Play back, compare against the original take, and automate parameters for evolving vocal effects.
